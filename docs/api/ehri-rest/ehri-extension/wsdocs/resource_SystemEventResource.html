SystemEventResource

Provides a web service interface for the Event model. Note: Event instances are created by the system, so we do not have create/update/delete methods here.

The following resources are applicable:

  • /systemEvent/{id}
  • /systemEvent/{id}/subjects
  • /systemEvent/byUser/{userId}
  • /systemEvent/for/{id}
  • /systemEvent/forUser/{userId}
  • /systemEvent/list
  • /systemEvent/versions/{id}
