CvocConceptResource

Provides a RESTful interface for the cvoc.Concept. Note that the concept creation endpoint is part of the VocabularyResource and creation without a Vocabulary is not possible via this API

The following resources are applicable:

  • /cvocConcept
  • /cvocConcept/{id}
  • /cvocConcept/{id}/broader/list
  • /cvocConcept/{id}/count
  • /cvocConcept/{id}/cvocConcept
  • /cvocConcept/{id}/list
  • /cvocConcept/{id}/narrower/{idNarrower}
  • /cvocConcept/{id}/narrower/list
  • /cvocConcept/{id}/related/{idRelated}
