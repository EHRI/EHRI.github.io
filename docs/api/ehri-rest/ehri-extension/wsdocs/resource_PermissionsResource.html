PermissionsResource

Provides a RESTful(ish) interface for setting PermissionTarget perms.

The following resources are applicable:

  • /permission
  • /permission/{userId}
  • /permission/{userId}/{id}
  • /permission/{userId}/scope/{id}
  • /permission/list/{id}
  • /permission/listForItem/{id}
  • /permission/listForScope/{id}

/permission

Mount Point: /permission

GET

Get the global permission matrix for the user making the request, based on the Authorization header.

Response Body

element: (custom)
media types: application/json

The current user's global permissions
