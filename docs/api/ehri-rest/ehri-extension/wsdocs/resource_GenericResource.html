GenericResource

Provides a means of fetching items and lists of items regardless of their specific type.

The following resources are applicable:

  • /entities
  • /entities/{id}
  • /entities/listByGraphId

/entities

Mount Point: /entities

GET

Fetch a list of items by their ID.

Parameters

name description type default
id A list of string IDs query

Response Body

element: (custom)
media types: application/xml
application/json

A serialized list of items

POST

Fetch a list of items by their ID.

Request Body

element: json
media types: application/json

A JSON-encoded list of string IDs

Response Body

element: (custom)
media types: application/xml
application/json

A serialized list of items

/entities/{id}

Mount Point: /entities/{id}

GET

Fetch an item of any type by ID.

Parameters

name description type default
id The item's ID path

Response Body

element: (custom)
media types: application/xml
application/json

A serialized representation.
