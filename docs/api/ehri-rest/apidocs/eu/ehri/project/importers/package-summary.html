Package eu.ehri.project.importers

Handlers and Importers and supporting classes to import files into the database.

See: Description

  • Interface Summary 
    Interface Description
    ImportCallback
    Implementing classes do things after an item was imported and created a mutation ('created', 'updated', 'unchanged').
  • Class Summary 
    Class Description
    AbstractImporter<T>
    Base class for importers that import documentary units, historical agents and virtual collections, with their constituent logical data, description(s), and date periods.
    EaHandler
    Base handler for EAD, EAC and EAG files, based on a SAX reader.
    EaImporter
    Import Map based representations of EAD or EAC for a given repository into the database.
    ImportLog
    Class that serves as a manifest for an import batch, detailing how many items were created and updated, and how many failed.
    MapImporter
    Importer of Map based representations of documentary units, historical agents, virtual collections and other entities.
    SaxXmlHandler
    Reader of XML files, creator of Map-based representations of EA* files.
