      • regenerateId

        public String regenerateId(String id,
                  boolean collisions,
                  boolean tolerant,
                  boolean commit)
                    throws ItemNotFound,
                           IOException,
                           IdRegenerator.IdCollisionError
        Regenerate the hierarchical graph ID for a given item, optionally renaming it.

        The default mode is to output items whose IDs would change, without actually changing them. The collisions parameter will only output items that would cause collisions if renamed, whereas tolerant mode will skip them altogether.

        The commit flag will cause renaming to take place.

        Parameters:
        id - the item's existing ID
        collisions - only output items that if renamed to the regenerated ID would cause collisions
        tolerant - skip items that could cause collisions rather then throwing an error
        commit - whether or not to rename the item
        Returns:
        a tab old-to-new mapping, or an empty body if nothing was changed
        Throws:
        ItemNotFound
        IOException
        IdRegenerator.IdCollisionError
