/tools/regenerate-id/{id}

Mount Point: /tools/regenerate-id/{id}

POST

Regenerate the hierarchical graph ID for a given item, optionally renaming it.

The default mode is to output items whose IDs would change, without actually changing them. The collisions parameter will only output items that would cause collisions if renamed, whereas tolerant mode will skip them altogether.

The commit flag will cause renaming to take place.

Parameters

name description type default
id the item's existing ID path
collisions only output items that if renamed to the regenerated ID would cause collisions query false
tolerant skip items that could cause collisions rather then throwing an error query false
commit whether or not to rename the item query false

Response Body

element: (custom)
media types: text/csv

a tab old-to-new mapping, or an empty body if nothing was changed
