    • eu.ehri.project.importers.AbstractImporter<T>

    Direct Known Subclasses:
    MapImporter

    public abstract class AbstractImporter<T>
extends Object
    Base class for importers that import documentary units, historical agents and virtual collections, with their constituent logical data, description(s), and date periods.
