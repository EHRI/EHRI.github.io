public abstract void importTopLevelExtraNodes(AbstractUnit topLevelUnit,
                                              Map<String,Object> itemData)
all data that is stored above the first imported DocumentaryUnit will be processed here, and added to the DocumentDescriptions. this might include MaintenanceEvents, or in the case of EAD there might be author/creation data.
