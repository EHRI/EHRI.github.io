Class Utilities

java.lang.Object
  eu.ehri.project.views.Utilities

public class Utilities
extends Object

A grab-bag of useful functions for making changes to the graph data. NB: Most uses of the methods here can be trivially accomplished with mutating Cypher but that's an even sharper tool. The purpose here is to formalise some very common operations with specific checks in place.
