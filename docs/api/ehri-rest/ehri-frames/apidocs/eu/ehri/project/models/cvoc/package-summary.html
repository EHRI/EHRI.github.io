Package eu.ehri.project.models.cvoc

Interface Summary
AuthoritativeItem An item that belongs in some authoritative set.
AuthoritativeSet A frame class representing a item that holds other authoritative items, such as concepts and historical agents.
Concept This models the thesaurus terms or keywords in a way that is better managing multi-linguality.
ConceptDescription labels, notes and texts that describe the concept, for a specific language are combined into this description NOTE: we should only allow one prefLabel (and thus one ConceptDescription) per language, but we cannot model that
Vocabulary A collection of 'related' concepts, or maybe a bit like the SKOS Concept Scheme Note that any concept in this Vocabulary that has no parent might be considered a topConcept.

AuthoritativeSet.Impl Implementation of complex methods.
Concept.Impl Implementation of complex methods.
