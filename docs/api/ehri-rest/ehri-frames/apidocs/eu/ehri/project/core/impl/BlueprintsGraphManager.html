public class BlueprintsGraphManager<T extends com.tinkerpop.blueprints.IndexableGraph>
extends Object
implements GraphManager

Implementation of GraphManager that uses a single index to manage all nodes. This class can be extended for when specific graph implementations (such as Neo4j) can provide more efficient implementations of certain methods.
