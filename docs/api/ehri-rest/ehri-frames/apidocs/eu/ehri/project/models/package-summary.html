Package eu.ehri.project.models

The interfaces in this package specify the classes of entities that map to graph frame classes.

See:
          Description

Interface Summary
Address A frame class representing an address.
Annotation A frame class representing an annotation.
ContentType Frame class representing a type of content in the graph.
Country Frame class representing a country.
DatePeriod Frame class representing a date period.
DocumentaryUnit A frame class for graph nodes representing documentary unit items.
DocumentDescription Frame class representing the description of a document.
Group Frame class representing a group of users or other groups that can be assigned permissions.
HistoricalAgent A frame class representing a historical agent item.
HistoricalAgentDescription Frame class representing the description of a historical agent.
Link Links two items together with a given body, with may either be a text property or some other entity.
MaintenanceEvent Frame class representing a pre-ingest event that took place upon some documentary unit item.
Permission Frame class representing a type of permission.
PermissionGrant Frame class representing a grant of a permission to a user.
Repository A frame class for graph nodes representing repository items.
RepositoryDescription Frame class representing the description of a repository.
UndeterminedRelationship Holds the information on a relationship specified in some Description, but without the target-end of the relationship being determined.
UnknownProperty Holds information about the collection or institute, but we don't specify (in the database) what it exactly means.
UserProfile A frame class representing a user within the database.
VirtualUnit Virtual documentary unit.
